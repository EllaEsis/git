cd desktop/git   进入目录
git init 初始化本地仓库
git add readme.txt  文件添加到索引(将修改添加到暂存区)
cd .. 回退
ssh-keygen -t rsa -C "...@qq.com"  生成密钥
绑定密钥 进入github -> settings -> ssh amd GPG keys -> add ssh key -> key (填入生成的id_rsa.pub) ->add key
git commit -m "..." 提交文件到本地仓库
git config --global user.email "...@qq.com" git配置用户邮箱
git config --global user.name "..."  git配置用户名
git remote add origin git@github.com:.../git.git 远程仓库源 
git push -u origin master 推送到远程仓库 -u关联了本地和远程的master分支
第一次使用git会提示ssh警告,用来验证key;请yes回车
git clone git@github.com:.../git.git 从远程克隆到本地
